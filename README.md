Prueba tenica (Productos Fullstack)

Este proyecto es una aplicacion fullstack para la gestion de productos, desarollada como parte de
una prueba tenica.


-Arquictectura 
  1-Frontend:Angular
  2-Backend-Springboot(ApiRest)
  3-Base de datos (Mysql/PostgreSQL)
  4-Control de versiones git
  5-

-Funcionalidades
  Listado de productos
  Creacion de productos
  Usuarios
  Autenticacion basica
  Comunicación frontend <> Backend mediante Api Rest
  Configuracion de CORS

REQUISITOS
  Java 17+
  Maven

  Node.js
  Angular CLI

EJECUCION EN ENTORNO LOCAL

  back
  cd backend
  mvn spring-boot: run:

  La API queda disponible en:
  hhtp://localhost:8080/api/v1/products

  Front
  cd frontend
  ng serve

  La aplicación quedará disponible en:
  hhtp://localhost:4200

CONFIGURACION CORS
  El backend tiene configurado el CORS para permitir peticiones desde el front

PRUEBAS
  Se hicieron pruebas mediante POSTMAN 

APRENDIZAJES CLAVE
  Importancia del estado de Git, como se comporta y como funcionan sus comandos.
  Correcta configuracion de endpoints y metodos HTTP
  Manejo de errores comunes como HTTP 405
  Conexion entre el back y el front.
  Manejo de seguridad mediante Cors o Cross.
  Mejor manejo de base de datos y implementacion en proyectos.
